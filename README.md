# Machine-Learning-Classification-Tasks
This repository contains Python scripts for four distinct machine learning classification tasks. Each script implements a different algorithm to solve a specific predictive modeling problem, complete with data preprocessing, model training, and performance evaluation.


## Repository Contents
Task 1: Breast Cancer Tumor Classification

File: task1.py

Algorithm: Random Forest Classifier

Dataset: Breast Cancer dataset (from sklearn.datasets)

Task 2: Diabetes Prediction (Logistic Regression) 

File: task2.py

Algorithm: Logistic Regression 

Dataset: Pima Indians Diabetes Dataset

Task 3: Titanic Survival Prediction

File: task3.py

Algorithm: XGBoost Classifier 

Dataset: Titanic Disaster Dataset

Task 4: Diabetes Prediction (Decision Trees) 

File: task4.py

Algorithm: Decision Tree Classifier 

Dataset: Pima Indians Diabetes Dataset


## How to Run
Clone this repository to your local machine.

Ensure you have the required libraries installed.

Run each script individually from your terminal or command prompt:

Bash

python task1_random_forest.py
